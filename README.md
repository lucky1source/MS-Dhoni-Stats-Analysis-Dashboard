# MS-Dhoni-Stats-Analysis-Dashboard
•MS Dhoni Stats Analysis Dashboard

An interactive cricket analytics dashboard built to analyze the career performance of Mahendra Singh Dhoni across ODI, T20, and Test formats.

-Project Overview

This dashboard provides a comprehensive statistical analysis of MS Dhoni’s international cricket career (2004–2019), including:

•Total Runs (ODI, T20, Test)

•Runs by Year Trend Analysis

•Runs by Batting Position

•Performance Against Top 10 Oppositions

•Top 5 Grounds (Runs Scored)

•Fours, Sixes & Not Outs by Format

•Strike Rate Comparison (ODI, T20, Test)

•Interactive Filters (Year & Opposition)

-Key Features

Dynamic filtering by year and opposition

Comparative format analysis

Clean and visually engaging layout

Data-driven cricket insights

-Purpose

This project demonstrates data visualization, dashboard design, and sports analytics skills by transforming raw cricket statistics into meaningful insights.
